# Bullet Echo Reference

A community-maintained technical reference for **Bullet Echo** and **Bullet Echo India**.

The goal of this repository is to provide a concise, factual and structured reference for the game's core mechanics, heroes, abilities and terminology.

This repository intentionally avoids:

* Hero tier lists
* Meta discussions
* Hero rankings
* Build recommendations
* Strategy guides
* Opinions

The purpose is to document **how the game works**, not **how to play it**.

## Scope

This repository documents gameplay mechanics that remain relatively stable across game updates.

Numerical values such as damage, cooldowns, durations, movement speed, health, armor and upgrade scaling may change between versions. Whenever a description states **"(depends on ability level)"**, it indicates that the exact value scales with upgrades.

---

## Naming Convention

Throughout this repository, the following terminology is used consistently.

* Main Ability
* Healing Ability
* Passive Ability (Slot 3)
* Passive Ability (Slot 4)
* Primary Weapon
* Secondary Weapon
* Hero Class
* Hero Faction

## Notes

* Information is written in a minimal, machine-readable format whenever practical.
* Bullet Echo India generally receives gameplay content later than the global version.
* Cosmetic content (skins, appearances, etc.) is outside the scope of this repository.
